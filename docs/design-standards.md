# 设计规范

## 颜色

| 用途 | 色值 | 说明 |
|------|------|------|
| 轨迹粉色 | `#FF69B4` (HotPink) | 手绘轨迹主色 |
| 提示文字 | `rgba(255,255,255,0.85)` | 半透明白色 |
| 提示背景 | `rgba(0,0,0,0.4)` | 半透明黑色底 |
| 工具栏背景 | `rgba(20,20,20,0.8)` | 深灰半透明 |
| 按钮主色 | `#FF69B4` | 粉色，与轨迹统一 |
| 彩色蒙版边缘 | 半透明白边框 | 3px 细线 + 12px 圆角 |

## 字体

| 用途 | 字体 | 大小 |
|------|------|------|
| 提示标题 | system-ui, sans-serif | 16px / 14px(移动端) |
| 按钮文字 | system-ui, sans-serif | 14px |

## Canvas 规范

| 参数 | 值 |
|------|-----|
| 最大分辨率 | 1280×720 |
| 移动端分辨率 | 640×480 |
| 轨迹默认粗细 | 4px |
| 轨迹粗细范围 | 2px–12px（随速度变化） |
| 彩色蒙版 feather | 3px（细边框，不过度羽化） |

## 布局

- 全屏 Canvas 容器，`position: fixed; inset: 0`
- 三层 Canvas 绝对定位叠放，尺寸完全一致
- 提示文字位于画面顶部 20px
- 工具栏位于画面底部 20px
- 响应式：max-width 100vw, max-height 100vh，object-fit: cover

## 动画

| 动效 | 时长 | 缓动 |
|------|------|------|
| 提示淡入淡出 | 300ms | ease |
| 工具栏按钮 hover | 150ms | ease |
| 手势触发反馈 | 200ms | ease-out |
